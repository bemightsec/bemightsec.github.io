DNS Enumeration Lab

A practical reconnaissance lab focused on DNS analysis, subdomain discovery, and information gathering using command-line tools.

Project 03 Reconnaissance DNS Analysis Enumeration

Project Overview

This lab demonstrates my practical understanding of DNS enumeration and early-stage reconnaissance. The goal was to gather public information about a target domain, inspect DNS records, and understand how domain infrastructure can be mapped during security assessments.

Tools Used

  • dig
  • nslookup
  • Subfinder
  • Basic DNS and TCP/IP knowledge

Focus Area

Reconnaissance, DNS record analysis, and information gathering as part of practical cybersecurity investigation workflow.

What I Did

  • Queried DNS records such as A, MX, TXT, and NS records
  • Performed reverse lookups on domains
  • Enumerated subdomains using command-line tools
  • Observed domain infrastructure and documented findings

Outcome

I successfully gathered useful DNS and subdomain information, improved my understanding of reconnaissance workflows, and became more confident using practical enumeration techniques in a lab environment.

What I Learned

This lab helped me understand how attackers and defenders can use public DNS information during the early stages of an assessment. It strengthened my command-line confidence and improved my ability to investigate digital infrastructure methodically.
